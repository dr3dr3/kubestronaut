# KCNA Study Materials

This folder contains all study materials, notes, and practice content for the Kubernetes and Cloud Native Associate (KCNA) certification.

### 01-kubernetes-fundamentals (46% of exam)
Core Kubernetes concepts, architecture, components, and objects. This is the largest domain and covers:

- Networking, storage, and configuration management

### 02-container-fundamentals (22% of exam)
Container technology concepts including:
- Container basics and benefits
- Container images and registries
- Container runtimes and standards (OCI)

### 03-cloud-native-architecture (16% of exam)
Cloud native principles and patterns:
- Microservices architecture
- 12-factor app methodology
- Autoscaling and self-healing
- Cloud native design patterns

### 04-cloud-native-observability (8% of exam)
Monitoring, logging, and tracing:
- Observability pillars (metrics, logs, traces)
- Prometheus and metrics
- Logging strategies
- Health checks and probes

### 05-cloud-native-application-delivery (8% of exam)
Application deployment and delivery:
- CI/CD concepts
- GitOps principles
- Deployment strategies
- Helm and package management
- Service mesh concepts

### practice-exams
Practice tests and exam preparation materials

### notes
Personal study notes, summaries, and cheat sheets
