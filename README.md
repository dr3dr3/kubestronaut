## Certifications Required

The Kubestronaut program requires passing the following certifications:
- [ ] Kubernetes and Cloud Native Associate (KCNA)
- [ ] Kubernetes and Cloud Native Security Associate (KCSA)
- [ ] Certified Kubernetes Application Developer (CKAD)
- [ ] Certified Kubernetes Administrator (CKA)
- [ ] Certified Kubernetes Security Specialist (CKS)

## Recommended Learning Path

The certifications are designed to be taken in a specific order to build knowledge progressively:

1. **[KCNA](KCNA-learning-plan.md)** (Jan 2026) - Foundation: Cloud native and Kubernetes concepts
2. **[KCSA](KCSA-learning-plan.md)** (Apr 2026) - Security fundamentals and best practices
3. **[CKAD](CKAD-learning-plan.md)** (Jul 2026) - Application development on Kubernetes (hands-on)
4. **[CKA](CKA-learning-plan.md)** (Oct 2026) - Cluster administration and operations (hands-on)
5. **[CKS](CKS-learning-plan.md)** (Nov 2026) - Advanced security specialist (hands-on, requires valid CKA)

### Certification Types
- **Knowledge-Based:** KCNA, KCSA (multiple choice, 90 minutes)
- **Performance-Based:** CKAD, CKA, CKS (hands-on terminal, 2 hours)
